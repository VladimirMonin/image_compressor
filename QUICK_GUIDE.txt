🆕 НОВОЕ В ЭТОЙ ВЕРСИИ: Безопасная обработка файлов!

═══════════════════════════════════════════════════════════

📁 НАСТРОЙКИ ОБРАБОТКИ ФАЙЛОВ:

┌─ 💾 Сохранять оригинальные файлы (рекомендуется!)
│  → При одинаковых форматах добавляется постфикс
│  → При разных форматах создаются оба файла
│
└─ 🗑️ Удалять оригинальные файлы (осторожно!)
   → При одинаковых форматах файл заменяется
   → При разных форматах старый удаляется

═══════════════════════════════════════════════════════════

🎛️ БЫСТРЫЕ НАСТРОЙКИ:

1️⃣ Безопасное сжатие (по умолчанию):
   ✅ Сохранять оригинальные файлы
   📝 Постфикс: _compressed

2️⃣ Замена файлов:
   ✅ Удалять оригинальные файлы

3️⃣ Свой постфикс:
   📝 Введите свой текст (например: _small)

═══════════════════════════════════════════════════════════

📋 ПРИМЕРЫ РЕЗУЛЬТАТОВ:

🔹 JPG → WebP (сохранять):
   photo.jpg ✓ остаётся
   photo.webp ✓ создаётся

🔹 JPG → JPG (сохранять):
   photo.jpg ✓ остаётся  
   photo_compressed.jpg ✓ создаётся

🔹 JPG → WebP (удалять):
   photo.jpg ✗ удаляется
   photo.webp ✓ создаётся

🔹 JPG → JPG (удалять):
   photo.jpg ✓ заменяется сжатой версией

═══════════════════════════════════════════════════════════

⚡ БЫСТРЫЙ СТАРТ:

1. Перетащите файлы/папки в окно
2. Выберите качество (80 = хорошо)  
3. Выберите формат (WebP = универсально)
4. Убедитесь: "Сохранять оригинальные файлы" ✅
5. Нажмите "Начать сжатие"

═══════════════════════════════════════════════════════════

🎯 ДЛЯ ПРОДВИНУТЫХ:

🔧 Массовая обработка: Перетащите целые папки
🎨 Форматы: HEIF (Apple), WebP (Google), AVIF (новый), JPEG (старый)
📊 Прогресс: Следите в реальном времени
📝 Логи: Детальная информация о каждом файле
⏹️ Остановка: Можно прервать в любой момент
